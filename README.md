# gruntme

> npm install the modules loaded in the gruntfile

## Install

```bash
npm i -g gruntme
```

## Usage
```
gruntme -S         install to dependencies
gruntme --save     install to dependencies
gruntme -D         install to devDependencies
gruntme --save-dev install to devDependencies
```
